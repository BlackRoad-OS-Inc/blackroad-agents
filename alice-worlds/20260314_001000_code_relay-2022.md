Certainly! Here's an interesting git trick: To find all branches that diverged from the main branch (e.g., `main` or `master`) by more than 5 commits, you can use the following command sequence:

```
git for-each-ref --format="%(refname)" refs/heads/* | \
while IFS= read -r ref; do
    if [ "$ref" != "refs/heads/main" ]; then
        if git merge-base main $ref > /dev/null 2>&1 &&
           ! $(git diff $ref..main --oneline | grep -q '^'); then
            echo $ref
        fi
    fi
done | sort -r -n -k 4 | head -n 5
```

This command finds and lists up to the last five branches that diverged from `main` by more than 5 commits. It's useful for quickly identifying which features or experiments have been actively worked on in parallel with the main branch.
