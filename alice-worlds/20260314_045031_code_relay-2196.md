Certainly! Here's a useful Git trick: 

To efficiently squash and amend the last few commits into a single commit (useful for history cleanup): `git reset <commit> --soft; git add -p; git commit -m "New commit message"`.

This command sequence allows you to modify the last few committed changes without rewriting the entire commit history, making it easier to keep your project's commit messages clean and consistent.
