ALICE: To quickly set up multiple authors for commits in your Git repository without manually changing the configuration each time, you can use the following alias in your `~/.gitconfig`:

```gitconfig
[alias]
  am <-author> = "!f() { if [ -n \"$2\" ]; then git commit --amend --reset-author --set=git://my-repo/$1/$2; else echo \"Usage: $0 <remote> <branch-name>\" >&2 ; fi}"
```

This alias allows you to easily amend commits with different authors by specifying the remote and branch name.
