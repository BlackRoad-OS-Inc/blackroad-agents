🚀 Quick Start — Local Ollama

1

Install Ollama

# macOS / Linux
curl -fsSL https://ollama.com/install.sh | sh

# macOS (Homebrew)
brew install ollama
2

Pull a Model

# Recommended: small and fast
ollama pull llama3.2

# For Pi / low-RAM devices
ollama pull qwen2.5:3b

# For heavy analysis
ollama pull qwen2.5:7b
3

Clone & Configure

git clone https://github.com/BlackRoad-OS-Inc/blackroad-agents.git
cd blackroad-agents
cp .env.example .env

# .env — point to your Ollama instance
OLLAMA_URL=http://localhost:11434
4

Run (TypeScript Registry)

npm install
npm run build
npm start   # starts registry on http://localhost:3001

# Run tests
npm test
5

Run (Python Agents)

pip install httpx

# Python — agents auto-detect Ollama
python3 -c "
import asyncio, os
os.environ['OLLAMA_URL'] = 'http://localhost:11434'
from src.agents import Lucidia

async def main():
    agent = Lucidia()
    reply = await agent.chat('What is the meaning of intelligence?')
    print(reply)

asyncio.run(main())
"
